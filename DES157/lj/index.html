Article # 2

Best Practices for Modals / Overlays / Dialog Windows | Naema Baskanderi

27 January, 2021

Naema Baskanderi writes in her article titled, Best Practices for Modals / Overlays / Dialog Windows, about the nature and the fruition of overlays. Overlays, modals, and dialog windows started as a band-aid approach to communicating with participants. There is typically a box with information, an input, perhaps some checkboxes, buttons for submission or navigating to the next screen, and the original home screen is just blurred, or darkened. Overlays have turned into the industry standard for how to deal with this issue, which provides accessability to the information through presentation. Communication is more clear and available to participants by using modals, overlays and dialog windows. it provides a way to, front and center, if you will, the information you are asking for from the user or wanting to display to them.

A little bit of what I've learned | 21 January, 2021

I am really enjoying this class so far, and this has actually been my favorite class so far at UCD. There is something magical that happens when what you're doing and what you love or enjoy to do are one in the same and this class is that for me. Something I learned from the Best practices for form design: 20 tips to boost form usability is that placeholder text should not reside within the input field, but below it. The less clutter and options the better. If there are 6 or less options in a drop-down menu, consider shifting to radio buttons or if there is multiple selections, best practices suggests using check boxes.
