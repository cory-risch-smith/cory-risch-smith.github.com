A little bit of what I've learned | 21 January, 2021

I am really enjoying this class so far, and this has actually been my favorite class so far at UCD. There is something magical that happens when what you're doing and what you love or enjoy to do are one in the same and this class is that for me. Something I learned from the Best practices for form design: 20 tips to boost form usability is that placeholder text should not reside within the input field, but below it. The less clutter and options the better. If there are 6 or less options in a drop-down menu, consider shifting to radio buttons or if there is multiple selections, best practices suggests using check boxes.

-Cory Risch-Smith
